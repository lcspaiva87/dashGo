## 🚀 Como executar

* Clone o repositório
* Instale as dependências com yarn
* Rode as migrations com yarn prisma db push
* Rode as seeds com yarn seed
* Inicie o servidor com yarn dev
* Agora você pode acessar localhost:3000 do seu navegador.
